"Librería Egg" - 2021

This project was an excercise from the Web Full Stack Developer course at Egg Educación institute.

It is a web management system for a library, where users can register and access every book available to apply for a loan. It also gives them the option to modify their profiles, and access a section where they can see records of every loan under their names (current and expired).


Login/register page

This is the first page everyone sees.




Users view

Homepage for regular users includes information like profile photo, user name and lastname, and register date.



Users view

Users also can see every book available to apply a loan.




Users view

Clicking "Know more" button displays full book's information, and lets the user apply to a loan.




User loans

This is the user's loans management section (active and old loans).




Administrator view

This role has management functions on users, books and loans.




User management dashboard

Displays all the information of every user registered, and also gives direct access to management functions (disable user, add a book loan for the user, view all loans of the user, change user's role, and delete user from database).


Books management dashboard

Displays all the information of every book registered, and also gives direct access to management functions (disable book, modify book information and delete book from database).



Loans management dashboard

Displays all the information of every loan registered, and also gives direct access to management functions (renew loan, register devolution or delete loan record from database).



Loans filtered by user

Administrators can also filter loans by user by clicking user name on Loans management dashboard.



The system also provides access to 'administrators': they have access to every book, user or loan registered in the database. Their functions are:

  • Manage books/authors/publishers (create, enable/disable, update, delete);
  • Manage registered users:
    • Enable / disable users;
    • Create books loans for them;
    • View every loan registered (also filtered by user);
    • Modify roles ('User' or 'Administrator');
    • Delete user from the database;
  • Manage loans (create, enable/disable, update, delete);

"Banquete San Jorge" - 2021

This is a personal project for a family banquet business called "Banquete San Jorge". The website has three important sections:

  • "Sobre nosotros" (about us): this section tells the story of the family business, from the origin to the present.

  • "Solicitá un Presupuesto" (request a budget): this is for me the most important and useful feature of the website. It's a form that filters the key information that is required to elaborate a budget on a service request. After the form is completed, the information is sent via e-mail to the person in charge so he can elaborate a budget and send it to the client.

  • "Contacto" (contact): this section provides contact information and direct links to WhatsApp chats.

In the last screenshot of the galery you can see that the e-mail sent from de web form has also a design made with css. The extra features are the "WhatsApp" and "Maps" buttons:

  • The "WhatsApp" button redirects to the customer's chat;
  • The "Maps" button redirects to Google Maps, searching the location provided by the client in the form.

The goal of these features is to simplify the task of preparing a budget: you have the key information of the event, access to the location of the event and the customer's contact information all in one place.
