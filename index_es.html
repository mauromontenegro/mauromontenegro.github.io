Mis Proyectos

"MTVDB" - 2022

Última actualización: 24/02/22

Este proyecto lo llevé a cabo por dos motivos: primero, poder aprender a consumir una API, y utilizar esa información para desarrollar un sitio web; en segundo lugar, me encanta ver películas y series, y siempre me interesa buscar información de lo que veo.

El sitio web brinda información detallada sobre Películas, Series y Shows de TV, incluyendo las personas que aparecen en ellos. Incluye la posibilidad de filtrar según categorías ('Populares', 'Mejor Puntuadas', etc.), y además tiene incorporado un buscador (que también tiene la opción de filtrar los resultados).

Página de Bienvenida

Es la primer página del sitio.

Vista de Categorías

Al filtrar Películas o Series por categoría, se muestran páginas con 20 cartas como máximo, además de indicar la cantidad de resultados y páginas de la categoría.

Vista de Categorías

Al final de cada página, existe un botón para volver hacia arriba, además de la navegación por páginas.

Vista de Detalles

Al hacer click en una carta (de película o serie), se muestra información más detallada sobre la misma, incluyendo géneros y los actores/actrices principales. También se incluyen tráilers de YouTube en caso de estar disponibles.

Vista de Detalles

Al hacer click en el botón de "Full cast...", nos dirige a la sección que nos muestra todos los actores/actrices que aparecen en la película o serie, incluyendo el nombre del personaje.

Vista de Actor/Actriz

Se detalla la información más importante sobre el actor/actriz. La placa que indica su edad cambia de color dependiendo de si la persona se encuentra con vida o no.

Vista de Actor/Actriz

Al clickear el botón de "Movies and TV Shows with...", se despliega la lista de créditos de la persona, ordenados del más reciente al más antiguo.

Buscador

Desde todas las vistas del sitio se puede acceder al botón de búsqueda en la barra de navegación. Desde el mismo se abre un cuadro con el espacio para ingresar la palabra clave buscada, y además incluye la posibilidad de filtrar la búsqueda por categoría (para obtener resultados más exactos).

Resultados de Búsqueda sin filtrar

Si no se selecciona una categoría al realizar una búsqueda, se devuelven los primeros 5 resultados de cada categoría para la palabra clave buscada.

Resultados de Búsqueda con filtro

En el caso de elegir un filtro, se listan todos los resultados para la palabra clave buscada.


El controlador consume la API de "The Movie Database", la cual me pareció excelente para practicar. La misma devuelve información con la estructura JSON; esa misma información la capturé utilizando 'RestTemplate' de Spring en Java.

El proyecto se realizó con el patrón MVC (Modelo Vista Controlador), utilizando Thymeleaf para generar plantillas reutilizables en el frontend. Una vez finalizado, el deploy lo hice con Heroku.

Por último, y no menos importante: el sitio web es full responsive. Se adapta a la perfección en dispositivos móviles.

¿Qué utilicé en el desarrollo?

Backend

  • Java 8
  • Spring framework
  • TMDB API

Frontend

  • HTML5/CSS3
  • Bootstrap v5.1
  • Thymeleaf

¿Dónde podés ver el proyecto?

El sitio web se encuentra en línea gracias a Heroku.

"Librería Egg" - 2021

Última actualización: 17/12/21

Este proyecto surgió como ejercicio del curso de 'Desarrollador Web Full Stack' en el Instituto Egg Educación.

Es un sistema de gestión web para una biblioteca, donde los usuarios pueden registrarse y acceder a todos los libros disponibles, y así solicitar un préstamo. Además, les da la opción de modificar su perfil, y acceder a una sección donde pueden visualizar información de cada préstamo registrado a su nombre (vigentes y vencidos).

Página de Login/registro

Es la primer página del sitio.

Vista de Usuarios

Página de inicio para usuarios comunes, que incluye información tal como foto de perfil, nombre y apellido, y fecha de registro.

Vista de Usuarios

Los usuarios también pueden ver cada libro disponible para solicitar un préstamo.

Vista de Usuarios

Al hacer click en el botón "Conocer más", se muestra información más detallada sobre el libro, y se le permite al usuario solicitar un préstamo del mismo.

Préstamos del Usuario

Esta es la sección de gestión de préstamos del usuario (activos y antiguos).

Vista de Administrador

Este rol tiene funciones de gestión de usuarios, libros y préstamos.

Dashboard de Gestión de Usuarios

Muestra toda la información de cada usuario registrado, y además brinda acceso directo a funciones de gestión (dar de baja el usuario, crear un préstamo para ese usuario, ver todos los préstamos del usuario, modificar el rol, y borrar el usuario de la base de datos).

Dashboard de Gestión de Libros

Muestra toda la información de cada libro registrado, y además brinda acceso directo a funciones de gestión (dar de baja el libro, modificar la información, y borrar el libro de la base de datos).

Dashboard de Gestión de Préstamos

Muestra toda la información de cada préstamo registrado, y además brinda acceso directo a funciones de gestión (renovar préstamo, registrar devolución o borrar el préstamo de la base de datos).

Préstamos filtrados por usuario

Los administradores también pueden filtrar préstamos por usuario, al hacer click en el nombre del usuario en el Dashboard de Gestión de Préstamos.


El sistema también brinda acceso a 'administradores': pueden acceder a cada libro, usuario o préstamo registrado en la base de datos. Sus funciones incluyen:

  • Gestionar libros/autores/editoriales (crear, dar de alta/baja, actualizar, eliminar);
  • Gestionar usuarios registrados:
    • Dar de alta / baja;
    • Crear préstamos en su nombre;
    • Ver todos los préstamos registrados (además de poder filtrarlos por usuario);
    • Modificar rol ('Usuario' o 'Administrador');
    • Borrar usuario de la base de datos;
  • Gestionar préstamos (crear, dar de alta/baja, actualizar, eliminar);

"Banquete San Jorge" - 2021

Última actualización: 17/12/21

Este es un proyecto personal realizado para la empresa de banquetes de mi familia, llamada "Banquete San Jorge". El sitio web tiene tres secciones importantes:

  • "Sobre nosotros": esta sección relata la historia del negocio familiar, desde sus orígenes hasta el presente.

  • "Solicitá un Presupuesto": esta es, a mi criterio, la característica más importante y útil del sitio web. Es un formulario que filtra la información clave requerida para elaborar un presupuesto ante una solicitud de servicio. Una vez completado el formulario, la información es enviada via e-mail a la persona encargada, así puede elaborar un presupuesto y enviárselo al cliente.

  • "Contacto": esta seccion provee información de contacto y enlaces directos al chat de WhatsApp.

Carousel

Sección de bienvenida al sitio.

Carousel

Además incluye imágenes con promociones y descuentos para eventos especiales.

Formulario 'Solicitá un presupuesto'

Los clientes pueden completarlo para enviar la información más importante sobre el evento.

Formulario 'Solicitá un presupuesto'

También pueden especificar qué artículos necesitan.

Email enviado a través del formulario web

Este es el email que la persona a cargo recibe, con toda la información solicitada en el formulario.


En la última imagen de la galería, podrás ver que el email enviado desde el formulario web tiene estilo, diseñado con css. Las características extra son el botón de 'WhatsApp' y el de 'Maps':

  • El botón de "WhatsApp" redirecciona al chat con el cliente;
  • El botón de "Maps" redirecciona a Google Maps, buscando la ubicación proporcionada por el cliente en el formulario.

El objetivo de estas características es simplificar la tarea de elaborar un presupuesto: la persona a cargo tiene la información clave del evento, acceso a la ubicación del mismo, y la información de contacto con el cliente, todo en un mismo lugar.
